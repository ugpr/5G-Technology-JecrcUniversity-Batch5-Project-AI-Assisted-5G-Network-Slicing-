# 5G-Technology-JecrcUniversity-Batch5-Project-AI-Assisted-5G-Network-Slicing-
AI-Assisted 5G Network Slicing using Reinforcement Learning is a MATLAB-based project that dynamically allocates bandwidth among eMBB, URLLC, and mMTC services using Q-Learning to optimize throughput, reduce delay, and improve overall 5G network performance.
